## pausemygame -- ahk 版本

本项目是 [`pausemygame`](https://github.com/cornradio/pausemygame) 的 AutoHotkey (AHK) 版本，可以暂停和恢复游戏。

### 功能简介

- 检测指定窗口是否处于活动状态
- 当窗口失去焦点时自动发送暂停指令
- 支持自定义热键和窗口名称

### 使用方法

1. 直接下载项目
2. 安装 AutoHotkey_1.1.37.02_setup.exe 
3. 双击 PauseManager.ahk 启动暂停工具
4. 快捷键 CTRL alt P 暂停游戏/程序
5. 快捷键 CTRL alt R 恢复游戏/程序
